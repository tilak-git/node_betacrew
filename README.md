1. **Requirements:** Node 16.17.0+
2. **Clone Repository:**
    ```bash
    $ git clone https://github.com/tilak-git/node_betacrew.git
    ```
3. **Navigate to the Project Directory**
    ```bash
    $ cd node_betacrew
    ```
4. **Start server**
    ```bash
    $ npm run server
    ```
5. **Start node client**
    ```bash
    $ npm run client
    ```
